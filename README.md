# Smart Attendance System

## Project Overview
The **Smart Attendance System** is a digital solution designed to streamline the process of tracking student attendance. This system allows teachers to mark attendance digitally, reducing manual errors and improving efficiency. 

## Team Information
- **Team Name:** SoloTech  
- **Team Member:** Sahad Chowdhury  

## Features
- Teacher login for attendance management
- Add students and assign them to specific courses
- Mark attendance manually or automatically (future enhancement)
- View attendance history and generate reports
- Store attendance records in a database for easy retrieval

## Technologies Used
- **Programming Language:** Java  
- **Framework:** JavaFX (for GUI) or Spring Boot (for web)  
- **Database:** MySQL or SQLite  
- **Development Tools:** IntelliJ IDEA / Eclipse, MySQL Workbench  

## UML Class Diagram (Initial)
### Main Classes:
1. **Student** – Represents students with attributes like ID, Name, and Course.  
2. **Course** – Stores course details such as Course ID, Name, and Instructor.  
3. **Teacher** – Manages attendance and course assignments.  
4. **AttendanceRecord** – Tracks student attendance status (Present/Absent) for each date.  
5. **ReportGenerator** – Generates attendance reports for analysis.

## Project Setup
### Prerequisites:
- Install **Java 17+**
- Set up **MySQL/SQLite** database
- Install **JavaFX**
